本專案是以MuscleCore 電商平台為情境，實作一個具備高擴充性、前後端分離的系統。 
此資料庫為專案前端客戶端，整合了消費者購物流程與商家管理端介面，專注於高效的資料渲染與使用者體驗。

## 🚀 技術亮點
* **TypeScript 全面導入**：定義商品 (Product)、訂單 (Order) 與使用者資料的 Interface，確保前後端 API 資料對接的一致性，大幅降低開發期的錯誤率。
* **商家管理後台 (Admin Dashboard)**：實作獨立的後台介面，供商家進行商品上架、即時資訊更新與庫存管理。
* **組件化架構設計**：透過 React 封裝高複用性的 UI 組件（如：商品卡片、管理表單），提升代碼維護效率。
* **響應式佈局 (RWD)**：確保消費者在行動裝置與桌機端皆能順暢完成購物流程。

## 🛠 主要功能
* **消費者端**：商品瀏覽、篩選搜尋、購物車清單管理。
* **商家後台**：
    * **商品 CRUD 管理**：支援商品新增、修改、刪除與列表檢視。
    * **即時監控**：商家可直覺式控管商品狀態與內容。
